### Enables Enterprise Policies, Auto-Update + Background update for Firefox.

Enables the following keys via .plist/Configuration Profile

 >AppAutoUpdate - true
 >
 >BackgroundAppUpdate - true
 >
 >EnterprisePoliciesEnabled - true
